## Usage

* Each day's problem has a corresponding folder 01/, 02/, ...
* In corresponding folder, both of the day's problems are solved by `python3 solve.py [/path/to/puzzle/input.txt]` with optional argument, default is `input.txt`.
* `input.txt` can be saved locally in a problem folder for convenience. With this name, it is ignored by git.
